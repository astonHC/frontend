# frontend
My intuitive Frontend for Applications

# MOTIVE:

The motive behind this project is to provide a means of which I am able to create frontend GUI apps.

This is by extension of me sorting out the technicalitiees of my Dissertation ahead of time.
By being able to create the tools I need in advance.

This frontend leverages SDL as the Media Layer for displaying Video, Images, Text, etc,
as well as imgui for the interface
